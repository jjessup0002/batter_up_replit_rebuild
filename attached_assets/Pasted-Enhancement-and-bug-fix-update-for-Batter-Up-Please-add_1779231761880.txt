Enhancement and bug-fix update for Batter Up.

Please add the following improvements to the current app while preserving the existing working features, navigation, styling, saved data, and overall user flow.

This update focuses on:

- Switching between Basic Mode and Advanced Mode during a game
- Editing the lineup during a game
- Returning to the Home screen without ending the game
- Fixing the Text Size setting
- Editing previous game stats from Game History
- Fixing the Setup Guide button in Settings
- Improving tutorials so they feel like guided walkthroughs instead of simple FAQ dropdowns

1. Allow Mode Switching During a Game

Add the ability for the user to switch between Basic Mode and Advanced Mode while a game is already in progress.

This does not need to be visible directly on the main Live Game screen.

Place it inside the in-game menu.

Suggested location:

Live Game Screen
→ Game Menu
→ Game Options
→ Switch Game Mode

The user should be able to switch:

- Basic Mode to Advanced Mode
- Advanced Mode to Basic Mode

When switching modes during a game:

- Do not delete game data
- Do not reset the lineup
- Do not reset the score
- Do not reset the inning
- Do not reset the current batter
- Do not reset balls, strikes, outs, or pitch count unless the user confirms that this is expected
- Preserve all recorded game events
- Preserve all player stats already recorded

When switching from Basic Mode to Advanced Mode:

- Keep the current game state
- Show advanced tracking tools going forward
- Allow detailed stats from that point forward
- Do not try to invent missing pitch-by-pitch data from earlier Basic Mode actions

When switching from Advanced Mode to Basic Mode:

- Keep the current game state
- Hide advanced controls from the main screen
- Keep advanced data in the background
- Do not delete previously recorded advanced stats
- Continue allowing the game summary to include whatever advanced data was already captured

Add a confirmation prompt:

Title:
Switch game mode?

Message:
You can switch modes without losing your game. Basic Mode keeps the screen simple. Advanced Mode shows more tracking options.

Buttons:
- Cancel
- Switch to Basic Mode
- Switch to Advanced Mode

Use only the button that applies if the current mode is known.

2. Edit Lineup During a Game

Add the ability to edit the lineup while a game is in progress.

Suggested location:

Live Game Screen
→ Game Menu
→ Edit Lineup

The coach should be able to:

- Add a player mid-game
- Remove a player from the active lineup
- Mark a player inactive
- Reorder players if needed
- Edit player name
- Edit jersey number
- Edit position

This is important because in youth sports, a player may show up late, leave early, get hurt, or be unavailable unexpectedly.

Adding a player mid-game:

When a player is added during a game, ask where to place them in the batting order.

Options:

- Add to bottom of lineup
- Insert after current batter
- Insert before current batter
- Choose position manually

Default recommendation:

Add to bottom of lineup

After adding the player:

- Save them to the current game lineup
- Ask whether to also save them to the original saved lineup

Prompt:

Player added to this game.

Would you also like to save this player to the saved lineup for future games?

Buttons:
- This game only
- Save to lineup too

Removing a player mid-game:

Do not permanently delete the player by default.

Use “Mark inactive for this game” as the safer default.

Prompt:

Remove player from active lineup?

This will remove the player from the batting rotation for the rest of this game, but it will not delete them from the saved lineup.

Buttons:
- Cancel
- Mark Inactive
- Delete From Saved Lineup

Only show “Delete From Saved Lineup” if it is safe and clearly intentional.

If the removed player is the current batter:

- Move current batter to the next active player
- Reset or preserve count based on the safest game logic
- If unsure, prompt the coach

If the removed player is on deck:

- Recalculate the on-deck batter

If a player is inactive:

- They should not appear as current batter
- They should not appear as on deck
- Their past stats from the game should remain saved
- They may still appear in the game summary with their recorded stats

Lineup editing must not break:

- Current batter tracking
- On-deck tracking
- Auto-scroll
- Score
- Inning
- Game events
- Game summary
- Player stats

Each troubleshooting topic should use this structure:

- What probably happened
- How to fix it now
- How to avoid it next time

Example:

Troubleshooting:
I can’t see the current batter

What probably happened:
The batting order list may be scrolled away from the highlighted batter.

How to fix it now:
Tap Next Batter or Previous Batter once, or use the current batter shortcut if available. Batter Up should automatically scroll to the current batter.

How to avoid it next time:
The app is designed to auto-scroll after batter changes. If it does not, return to the game and try changing batters again.
